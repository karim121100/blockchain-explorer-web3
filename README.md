# Blockchain Explorer (Web3)

This project simulates a simple blockchain explorer.

It helps beginners understand how blocks, transactions, and wallets are viewed on a blockchain.

## What this project does

- Creates blocks
- Stores transactions
- Tracks wallet activity
- Shows blockchain data

## Why this project is useful

You will learn:
- How blocks are structured
- How transactions are linked
- How explorers read blockchain data
- How analytics systems work

## Tech used

- JavaScript
- Node.js
- Explorer concepts
- Blockchain logic (simulated)

## How to run

1. Install Node.js  
2. Clone this repo  
3. Open terminal in project folder  
4. Run: `node app.js`  

## Files overview

- `app.js` → Main app file  
- `block.js` → Block logic  
- `transaction.js` → Transaction logic  
- `wallet.js` → Wallet tracking  
- `chain.js` → Blockchain storage  
- `analytics.js` → Data analysis  
- `config.js` → Settings  
- `utils.js` → Helpers  
- `logger.js` → Logs  
- `package.json` → Project info

## Notes

This project is for learning only.  
Do not use this as a real blockchain explorer.
